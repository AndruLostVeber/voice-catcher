# Changelog

Все заметные изменения этого проекта.

## v0.18 — Семантический поиск
- Семантический поиск истории через NVIDIA `nv-embedqa-e5-v5` embeddings
- Cosine similarity + in-memory cache embeddings
- Скор показывается рядом с заголовком сессии

## v0.17 — Переанализ
- Кнопка «🔄 Переанализ» для существующей сессии
- Пересчёт саммари и глубокого анализа другой моделью без перезаписи аудио
- Параллельный пересчёт через ThreadPoolExecutor

## v0.16 — Форматирование времени
- Длительности отображаются как `mm:ss` или `h:mm:ss`
- Везде, где раньше был `Xс`

## v0.15 — Переоткрытие сессии
- Кнопка «🔄 Открыть» в истории
- Загружает выбранную сессию обратно в основной вид с графиками и аналитикой

## v0.14 — Welcome state
- Пустая история теперь показывает 3 карточки с подсказками для сценариев

## v0.13 — Open folder
- Кнопка «📂 Открыть папку экспортов» в проводнике (Windows/macOS/Linux)

## v0.12 — Выбор языка распознавания
- Сайдбар: auto / ru / en / uk / de / es
- По умолчанию ru

## v0.11 — README
- Полная карта фич, диаграмма архитектуры, badges

## v0.10 — Bulk-экспорт
- Скачать всю историю одним ZIP (Markdown + JSON)

## v0.9 — Аналитика истории
- Bar chart минут по дням
- Pie chart типов сессий
- Area chart часов суток

## v0.8 — Оптимизация
- Параллельная транскрипция двух дорожек
- Параллельные LLM-вызовы (summarize + deep_analyze)

## v0.7 — UI/UX
- Кастомный CSS (тёмная тема, градиенты, скругления)
- Пульсирующий красный индикатор во время записи
- Тёмная тема в `.streamlit/config.toml`
- Дополнительные модели: Llama 3.3, Llama 4 Maverick, Nemotron Super 49B, Mixtral, GPT-OSS

## v0.6 — Визуализация
- Pie chart баланса времени речи (Altair)
- Bar chart слов и WPM по сторонам
- Line chart эмоций по фазам

## v0.5 — Поиск в истории
- Текстовый поиск
- Фильтры по типу, сортировка
- Общие метрики

## v0.4 — Автосохранение Markdown
- После каждой обработки сохраняется .md в `data/exports/`

## v0.3 — Windows-уведомления
- plyer + winsound при готовности обработки
- Toggle в сайдбаре

## v0.2 — Глубокий анализ
- `src/analyzer.py`: локальная talk-статистика + LLM deep_analyze
- Стиль общения, эмоциональный таймлайн, цитаты, рекомендации, риски

## v0.1 — Базовые фичи (нулевой день)
- Запись с микрофона (sounddevice)
- Запись звонка через WASAPI loopback (PyAudioWPatch)
- Диаризация Я / Собеседник
- Транскрипция faster-whisper
- Саммари через NVIDIA Llama 3.3 70B
- Экспорт в Markdown
- История сессий в JSON
- Audio-плеер в результатах и истории
- Streamlit UI с 4 вкладками
